# Resize and compress images
simple script to resize and compress images with nodejs.


<h1>Dependencies</h1>
<ul>
<li>sharp</li>
<li>gifsicle</li>
<li>pngquant-bin</li>
<li>compress-images</li>
</ul>

<h2>Usage</h2>

Use nodejs to run app.js with 3 arguments.

<code>node path_of_file width height</code>
